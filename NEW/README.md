# SynthRare-Longitudinal
Causal, multimodal synthetic patient trajectories for rare disease research using DiffXGAN.

- 10K patients (EHR + CT + reports)
- Causal annotations: "If platelets >150k at t=3, survival ↑42%"
- DiffXGAN: Diffusion + XAI + Privacy (ε < 0.5)
- NeurIPS 2025 submission
